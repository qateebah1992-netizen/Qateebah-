# تطبيق كيان برو - متجر إلكتروني

## نظرة عامة
تطبيق أندرويد للتجارة الإلكترونية يحتوي على واجهات حديثة وسهلة الاستخدام.

## المميزات

### 1. شاشة الترحيب (Splash Screen)
- عرض شعار التطبيق
- انتقال تلقائي إلى شاشة تسجيل الدخول بعد 3 ثوانٍ
- تصميم جذاب بألوان التطبيق الأساسية

### 2. شاشة تسجيل الدخول (Login)
- إدخال البريد الإلكتروني وكلمة المرور
- التحقق من صحة البيانات
- حفظ حالة تسجيل الدخول
- رابط للانتقال إلى شاشة التسجيل

### 3. شاشة التسجيل (Register)
- إدخال الاسم الكامل
- إدخال البريد الإلكتروني
- إدخال كلمة المرور وتأكيدها
- التحقق من تطابق كلمات المرور
- حفظ بيانات المستخدم

### 4. شاشة المتجر الإلكتروني (Main Store)
- عرض ترحيبي بالمستخدم
- بانر عروض خاصة
- عرض المنتجات في شبكة (Grid)
- بطاقات منتجات احترافية
- زر إضافة للسلة لكل منتج
- قائمة خيارات مع إمكانية تسجيل الخروج

## البنية التقنية

### الأنشطة (Activities)
1. **SplashActivity**: شاشة البداية
2. **LoginActivity**: شاشة تسجيل الدخول
3. **RegisterActivity**: شاشة إنشاء حساب
4. **MainActivity**: الشاشة الرئيسية للمتجر

### النماذج (Models)
- **Product**: نموذج بيانات المنتج (الاسم، السعر، الصورة)

### المحولات (Adapters)
- **ProductAdapter**: محول عرض المنتجات في RecyclerView

### التخزين
- استخدام SharedPreferences لحفظ:
  - بيانات المستخدم
  - حالة تسجيل الدخول
  - معلومات الجلسة

## المكتبات المستخدمة
- **AndroidX AppCompat**: التوافق مع الإصدارات القديمة
- **Material Components**: مكونات Material Design
- **RecyclerView**: عرض القوائم بكفاءة
- **CardView**: بطاقات المنتجات

## متطلبات التشغيل
- Android SDK 21 أو أحدث (Android 5.0+)
- Android Studio Arctic Fox أو أحدث
- Gradle 7.0+

## خطوات التشغيل

1. افتح Android Studio
2. اختر File > New > Import Project
3. حدد مجلد المشروع
4. انتظر حتى يتم بناء المشروع
5. قم بتشغيل التطبيق على محاكي أو جهاز حقيقي

## ملاحظات التطوير

### للتطوير المستقبلي:
1. **قاعدة بيانات**: ربط التطبيق بقاعدة بيانات حقيقية (Firebase / SQLite)
2. **الصور**: استخدام مكتبة Glide أو Picasso لتحميل الصور
3. **API**: التكامل مع API خلفي للمنتجات
4. **سلة المشتريات**: إضافة وظيفة سلة مشتريات كاملة
5. **الدفع**: دمج بوابة دفع إلكتروني
6. **التنبيهات**: إضافة نظام إشعارات
7. **البحث**: إضافة وظيفة البحث عن المنتجات
8. **الفئات**: تصنيف المنتجات حسب الفئات
9. **الملف الشخصي**: صفحة ملف شخصي للمستخدم
10. **المفضلة**: إضافة منتجات للمفضلة
